# Golden Cocoa

> **Pure Taste, Pure Energy** — Premium 100% natural cocoa powder from Cameroun, delivered across Mauritius.

A modern, premium e-commerce single-page website for **Golden Cocoa** — showcasing the Gol Powder Cacao product line with an elegant luxury design.

## Features

- Fully responsive (mobile + desktop)
- Luxury design — dark brown, gold, cream palette
- Embedded product video showcase
- Two product packs with highlighted pricing (200g · Rs 350 / 400g · Rs 650)
- WhatsApp & Facebook integration with pre-filled order messages
- Floating "Chat on WhatsApp" CTA
- Customer reviews section
- Island-wide delivery information
- Contact form that redirects to WhatsApp
- Smooth scroll animations & reveal effects
- SEO optimized with Open Graph meta tags

## Tech Stack

Pure HTML, CSS, and JavaScript — no build step, no dependencies. Fast loading, deploy anywhere.

## Deploy on Vercel

1. Push to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repository
4. No configuration needed — Vercel auto-detects the static site
5. Deploy

## Local Preview

```bash
python3 -m http.server 3000
# or
npx serve .
```

Then open http://localhost:3000
